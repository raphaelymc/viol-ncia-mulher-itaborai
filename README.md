## 📊 Sobre o projeto

Este projeto foi desenvolvido com foco no município de **Itaboraí (RJ)**, com o objetivo de apresentar de forma clara e acessível dados relacionados à violência.

A proposta surgiu da necessidade de transformar dados públicos — muitas vezes difíceis de acessar — em uma visualização simples, compreensível e útil para a população.

Os dados utilizados foram obtidos a partir do **Instituto de Segurança Pública do Estado do Rio de Janeiro (ISP-RJ)**, com base em registros oficiais.

## 📌 Sobre os dados

Os dados utilizados são provenientes do **Instituto de Segurança Pública do Estado do Rio de Janeiro (ISP-RJ)**.

- Os dados são públicos e oficiais  
- Foram organizados manualmente para análise  
- Alguns indicadores não possuem detalhamento completo sobre o perfil das vítimas  

Por isso, a interpretação deve ser feita com responsabilidade.

---

## 📊 Tratamento dos dados

Neste projeto, optei por utilizar **Excel** para organizar e estruturar os dados públicos.

Essa escolha foi suficiente para o volume de dados utilizado e permitiu focar na análise e na construção da solução em nuvem.
